Instruções de uso

Para acessar o cadastro, basta abrir a pasta "cadastro", e abrir o arquivo "index.html".

Para efetivar o cadastro, o usuário deve digitar o Nome da Loja, E-mail, Senha e fazer uma descrição da loja. Além disso, deve selecionar
a categoria da loja que vai ser cadastrada, através do botão posicionado abaixo de "senha". Ao fim do cadastro, o usuário deve clicar no 
botão "Registrar" APENAS UMA VEZ para finalizar o registro.
